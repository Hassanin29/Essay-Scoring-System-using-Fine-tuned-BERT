# Essay-Scoring-System-using-Fine-tuned-BERT
صممت نموذج تصحيح تلقائي للمقالات بدقة 89.5% F1-score و MSE 0.11 على 1700 مقالة متنوعة (التقييم من 1 لـ 6).  طوّرت معالجة للنصوص الطويلة (أكثر من 512 token) باستخدام sliding window with stride لاستيعاب المقالات الكاملة دون فقدان السياق.  طبّقت FP16 quantization لتقليل حجم الموديل من 418 ميجا إلى 209 ميجا مع الحفاظ على دقة 90%،
